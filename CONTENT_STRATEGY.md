## 3. CONTENT STRUCTURE & SEO OPTIMIZATION

### Blog Post Template

```markdown
# [Main Keyword]: [Benefit/How-To]

## Introduction (100-150 words)
- Hook the reader
- Explain relevance
- Preview main points
- Include target keyword

## Section 1: [Subtopic] (200-300 words)
- Detailed explanation
- Examples
- Internal links
- Related keywords

## Section 2: [Subtopic] (200-300 words)
- Practical information
- Tips and tricks
- Statistics/data
- External sources

## Section 3: [Subtopic] (200-300 words)
- Advanced information
- Best practices
- Common mistakes
- Solutions

## Conclusion (100-150 words)
- Summary of key points
- Call to action
- Related resources
- Next steps

## FAQ Section (200-300 words)
- 3-5 common questions
- Concise answers
- Links to related content

## Related Resources
- Internal links to related posts
- Links to guides
- Links to tools
```

### SEO Optimization Checklist for Each Post
- [ ] Target keyword in title (front-loaded)
- [ ] Target keyword in first 100 words
- [ ] Target keyword in headings (H2, H3)
- [ ] 2-3 related keywords throughout
- [ ] Meta description (150-160 characters)
- [ ] Internal links (3-5 minimum)
- [ ] External links (2-3 authoritative sources)
- [ ] Images with alt text
- [ ] Proper heading hierarchy
- [ ] Readability score: 60+
- [ ] Word count: 1000+ (blog posts)
- [ ] Schema markup (Article schema)
